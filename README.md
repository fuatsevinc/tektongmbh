## About the Project

This project is a professional corporate website developed for TEKTON GmbH, a company operating in the field of civil engineering and network infrastructure. The site presents the company's services, past projects, team, and contact information in a clean and modern layout optimized for all device sizes.

---

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll navigation with sticky header
- Hero section with animated background
- Service showcase sections (Tiefbau, Glasfaser, Projektmanagement, Standortplanung, Qualitaetskontrolle)
- Project portfolio gallery
- Dedicated Team section
- Contact form with PHP backend
- Saugbagger (vacuum excavator) dedicated page
- Elektrotechnik (electrical engineering) dedicated page
- GitHub Pages deployment
